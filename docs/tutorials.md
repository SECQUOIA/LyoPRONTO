# Tutorials

The hosted web GUI and original video tutorial remain useful orientation:

- Web GUI: <http://lyopronto.geddes.rcac.purdue.edu>
- Video tutorial: <https://www.youtube.com/watch?v=DI-Gz0pBI0w>

For the current repository examples, install the package in editable mode and
run a maintained example from the repository root:

```bash
python -m pip install -e ".[dev]"
python examples/example_web_interface.py
```

For the file-oriented compatibility path, edit `main.py` and run:

```bash
python main.py
```

Typed Pint API examples can be run with:

```bash
python -m examples.typed_api_examples
```
