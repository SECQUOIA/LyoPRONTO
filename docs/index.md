# LyoPRONTO Documentation

LyoPRONTO is a Python package for vial-scale lyophilization simulation and
optimization. It includes legacy dictionary-based calculators, maintained
script examples, an additive typed Pint API, and optional Pyomo prototypes.

- [User Guide](how-to-guides.md): setup, examples, notebooks, local validation,
  and legacy workflow notes.
- [API Reference](reference.md): package map, API boundaries, unit conventions,
  and optional Pyomo model status.
- [Physics Reference](technical/physics-reference.md): model equations and
  scientific assumptions.
- [Julia Parity](technical/julia-parity.md): typed API parity notes.
- [Contributor Guide](dev.md): CI lanes, branch-protection policy,
  documentation builds, and maintenance rules.
